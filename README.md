# Motorsport Electronics

> This is an unofficial community project and is not affiliated with or endorsed by any manufacturer or vendor.

A community library for sharing motorsport electronics files, configurations, and related assets.

The aim of this repository is to make it easier for people to share, discover, improve, and reuse useful files across the motorsport electronics community.

## What this repo is for

This repository is intended for:

- dash files and layouts
- CAN configurations and templates
- ECU maps and related configuration files
- PMU / PDM configurations
- display assets
- screenshots
- setup notes and supporting documentation
- other useful motorsport electronics files

This is intended to be a practical shared library, not a dumping ground.

## Contributing

Contributions are welcome.

There are two ways to submit files:

### 1. Pull request

Best option if you are comfortable with GitHub.

- Fork the repository
- Add your files in the most appropriate folder
- Commit your changes
- Open a pull request

### 2. Submission issue

Best option if you just want to upload a file without using Git.

- Open a new submission issue
- Fill in the template
- Attach the file and screenshots where relevant
- A maintainer will review it and add it to the library if suitable

Please read [CONTRIBUTING.md](CONTRIBUTING.md) before submitting.

## Repository structure

Example top-level structure:

- `Dash/`
- `CAN/`
- `ECU/`
- `PDM_PMU/`
- `Displays/`
- `Docs/`
- `Other/`

Subfolders may be organised by:

- manufacturer or platform
- hardware model
- vehicle or engine platform
- road / race / rally / drift / endurance use
- light / dark theme
- verified / unverified status

The structure may evolve as the library grows.

## What to include with a submission

Where possible, please include:

- file name and short title
- target hardware or software
- firmware or software version
- manufacturer or platform
- short description of what the file does
- screenshot where relevant
- setup notes
- any dependencies such as fonts, icons, images, CAN databases, or external assets

## Rules

Do not upload:

- files you do not have permission to share
- commercial or paid assets you are not allowed to redistribute
- copyrighted fonts, logos, graphics, or backgrounds without permission
- confidential customer or team files
- malicious, misleading, or unsafe files

## Important note

Files in this repository are shared by the community.

Their presence here does **not** guarantee that they are:

- tested
- safe
- complete
- suitable for your setup

Always review imported files carefully before using them on a live vehicle.

## Verified content

Some files may later be marked as **verified** where a maintainer or contributor has confirmed basic details such as hardware target, version compatibility, or successful use.

Unless marked otherwise, assume content is community-submitted and unverified.

## Goals for this project

- make useful motorsport electronics files easier to find
- reduce duplicated effort
- encourage people to improve and adapt existing work
- preserve useful community knowledge in one place
